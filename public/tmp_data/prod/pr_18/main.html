Технология производства

  1. Армирование: пространственные каркасы свариваются роботизированными комплексами «KUKA» с точностью ±0.5 мм.
  2. Бетонирование: заливка в виброформы с уплотнением на частотных станциях для исключения пустот.
  3. Пропаривание: 12-часовая термообработка при 80°C для достижения 70% прочности за сутки.
  4. Контроль: испытания на прессе до 3000 тонн и ультразвуковая дефектоскопия каждой партии.
